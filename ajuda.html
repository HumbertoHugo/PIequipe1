Segurança no site

Identificando uma página falsa

Sites falsos têm estado cada vez mais parecidos com os originais, por isso, pode ser muito difícil diferenciá-los visualmente. vamos aprender juntos?

A url do site: A primeira coisa é reparar na url do site e observar também o cadeado na barra de pesquisa. caso esteja destrancado, saia imediatamente.

A quantidade de anúncios: Embora seja uma fonte de renda para muitos sites, é pouco provável que um site confiável exceda as propagandas na página e, principalmente, se essas forem invasivas (como é o caso de pop-ups). Observe o conteúdo dos anúncios, isso pode fazer toda diferença! se estiverem fora de contexto ou não associados às suas pesquisas recentes, pode haver algo de errado.

semelhante, mas é idêntico?: Normalmente, os sites que “clonam” as versões originais possuem detalhes que se diferenciam, pois não acompanham as atualizações ou não possuem o mesmo nível de programação/design da página. Por isso, caso já tenha acessado o site anteriormente, vale a pena checar mais uma vez os detalhes da página. Caso encontre algum dos pontos já citados, levante sua bandeira vermelha de atenção.

Pesquise o site na página do Google status: Busque o nome do site na página de status do google. Isso te ajuda a analisar o nível de transparência de determinado link. 

Velocidade de carregamento: Os sites falsos muitas vezes são hospedados em servidores de baixa qualidade. Caso o site original tenha uma estrutura mais ágil, é fácil reparar a diferença de qualidade de uma versão para a outra.


Transação de dados

A segurança nas transações é uma das nossas grandes preocupações, por isso, no PlayandBuy, todos os dados são criptografados, garantindo a proteção dos pedidos. Outras medidas também são tomadas: a transferência do pagamento para o vendedor é feita somente depois de o comprador confirmar o recebimento e, em casos de reembolso, o comprador só recebe o valor reembolsado quando o vendedor estiver com o produto em mãos. É claro, o enjoei acompanha todas as etapas do processo.
